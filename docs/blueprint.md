# **App Name**: TraceFlow Hub

## Core Features:

- Live Factory Dashboard: Real-time summary with KPI cards and part-wise visual indicators using Green/Yellow/Red status logic for stock and dispatch completion.
- Guided Morning Stock Entry: Simplified mobile-friendly forms for entering PDI and inspection stock for all parts with automatic available stock calculation.
- Vehicle Dispatch Module: Three-vehicle concurrent loading interface with automatic summation of part quantities and instant balance updates.
- Intelligent Shortage Forecast: A tool that uses generative AI to analyze historical dispatch trends and current stock levels to predict potential shortage risks 48 hours in advance.
- Inventory Traceability Engine: Maintains an immutable ledger of stock revisions and vehicle loading timestamps for compliance and audit trails.
- Automated Risk Alert Panel: Top-level dashboard section highlighting only high-risk parts where planned dispatch exceeds current balance stock.
- Reporting & Export Suite: Generates Daily Stock and Dispatch reports with instant export capabilities to PDF and Excel formats.

## Style Guidelines:

- Primary Color: Vibrant Cobalt (#3273FF) representing precision and flow; used for primary actions and key navigation elements.
- Background Color: Industrial Dark Navy (#10141D), providing high contrast for factory floor visibility under various lighting conditions.
- Accent Color: Electric Cyan (#00E5FF) used for critical system feedback and focus states on data entry fields.
- Headline Font: 'Space Grotesk' (sans-serif) for a modern, tech-focused look. Body Font: 'Inter' (sans-serif) for maximum legibility in complex data tables.
- High-stroke, minimalist geometric icons for warehouse actions, using thick lines for visibility on mobile screens during fast-paced movement.
- Factory-first design with extra-large tap targets (min 48px), high-contrast data rows, and sticky progress bars for each part entry.
- Smooth width transitions for dispatch progress bars and gentle pulse effects for 'Stock Not Available' status indicators.
