
# Kalyani Dispatch Stock Tracker

Real-time stock traceability and dispatch planning system for modern factory floors.

## Core Features
- **Multi-System Real-Time Sync**: Powered by Firebase Firestore. Changes reflect instantly on all connected PCs and mobile devices.
- **Strict Frozen Headers**: Summaries and table controls are permanently locked at the top of the viewport.
- **Ultra-Fast Hydration**: 0.1s - 0.2s synchronization experience.
- **Professional Factory UI**: High-impact typography with expanded character spacing.
- **Theme Support**: Professional Dark and Light modes optimized for industrial lighting.
